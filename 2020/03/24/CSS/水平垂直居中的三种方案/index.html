CSS/水平垂直居中的三种方案

Posted on 2020-03-24

单行文本的水平垂直居中

text-align: center; line-height: 父元素的高度

总结关于一个width: 100px; height: 100px; background: orange;在它的包含块的水平垂直居中

position 3种(注意的是，父元素要设置position: relative;作为参照)

  • postition: absolute; left: 0; right: 0; top: 0; bottom: 0; margin: auto;
  • position: absolute; left: 50%; top: 50%; margin-left: -1/2宽; margin-top: -1/2高;
  • position: absolute; left: 50%; top: 50%; transform: translate(-50%, -50%)

弹性盒子

  • 父元素: display: flex; justify-content: center; align-items: center;
