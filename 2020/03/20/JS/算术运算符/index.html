算数运算符

  • % 得到的值的符号主要看前一个数的符号 如 A % B 得到的符号主要看 A 的符号

-除加号外的其他算术运算符号 - * / % 等 ， 在其他类型的运算时，即string boolean undefined null 对象时，会先将他们转为number类型，再运算 null => 0 undefined => NaN 对象 => NaN

  • 加号

#两边至少有一边是字符串，则得到的是字符串拼接的结果

number -> 数字字符串
null -> “null”
underfined -> “undefined”
对象 -> “[object Object]”

#两边都不是字符串，但是一边是对象，先将对象转为字符串，在进行字符串的拼接

#其他按照正常数学运算进行运算

+{} a + {} 的结果不一样 + {} 表示将对象转为 number类型的 NaN a + {} 先将{} 转为字符串，再按照字符串拼接 “a[object Object]”

  • “” / “ “ 转为number都为0

-0 / 0 ==> NaN

-+”” ==> 表示正的意思， 转为number
