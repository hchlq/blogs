CSS/水平垂直居中总结

Posted on 2020-03-22

css的水平垂直居中总结(都是有包含块的前提之下)

浮动元素没有水平垂直居中

水平居中：

  • 单行文本的水平居中：在其父元素加 text-align: center;
  • 常规流块盒的水平居中： 本身使用 已知宽 +margin: 0 auto;(常规流块盒不可以垂直居中)
  • 定位元素的水平居中（absolute 和 fixed） left:0 right:0 margin: 0 auto

垂直居中：

  • 单行文本： 设置 line-height: 等于父级的 height
  • 多行文本： 在文本再包上容器，让该容器变为定位 或者文本的父级使用 padding 设置成垂直居中的样式
  • 定位元素的垂直居中：
    一 定高 top : 0 bottom: 0 margin: auto 0;
    二 定高 top: 50% margin-top: -height;
    三 定高 top: 50% transform: translate(0, -50%); 高度不确定也行，高度可以任意改变

目前关于定位元素的水平垂直居中的三个方案

要定死宽高的 height: 200px; width: 200px; position：absolute; top: 50%; left:50%; margin-left: - 100px(二分之一宽); margin-top: -100px（二分之一高）;

宽高自适应的，不用定死 可以自定义 ：
1 position: absolute; left: 0; bottom: 0; left: 0; right: 0; margin: auto;

2 position: absolute; left: 50%; top: 50%; transform: translate(-50%, -50%)
